# Online-Book-Store-
Online Book Store Management System | SQL Project

Project Description:
1. Designed and developed an Online Book Store Management System using SQL to manage and analyze bookstore data efficiently.
2. Created and worked on multiple relational database tables including Books, Customers, and Orders.
3. Performed data analysis to generate meaningful business insights related to sales, inventory, customers, and book categories.
4. Implemented SQL queries to retrieve, filter, sort, and analyze bookstore records.
5. Analyzed customer purchasing patterns, book demand, and revenue generation.
6. Calculated inventory stock and identified books with low availability.
7. Generated reports for genre-wise sales, customer orders, and total revenue.
8. Used aggregate functions, joins, subqueries, and grouping operations for advanced data analysis.
9. Improved database querying and problem-solving skills through real-world business scenarios.
10. Demonstrated practical understanding of database management and SQL-based business analytics.
